# 🌈 VynixShader

**VynixShader** is a lightweight Minecraft shader focused on delivering beautiful visuals while maintaining smooth performance. It enhances lighting, skies, water, shadows, and colors without sacrificing FPS, making it perfect for Survival, Creative, and PvP.

---

## ✨ Features

* 🌅 Enhanced lighting and shadows
* ☀️ Beautiful sunrises and sunsets
* 🌊 Realistic water reflections
* ☁️ Improved clouds and sky rendering
* 🌿 Vibrant, natural color grading
* 🌙 Better nighttime atmosphere
* ⚡ Optimized for high FPS
* 🎮 Great for Survival, Creative, and PvP

---

## 📥 Installation

### Iris Shaders (Recommended)

1. Install Iris Shaders.
2. Download the latest **VynixShader** release.
3. Open Minecraft.
4. Go to **Options → Video Settings → Shader Packs**.
5. Drag the downloaded shader ZIP into the shader packs folder or click **Open Shader Pack Folder** and place it there.
6. Select **VynixShader** and click **Apply**.

### OptiFine

1. Install OptiFine (if your version supports shaders).
2. Place the shader ZIP into the `shaderpacks` folder.
3. Enable **VynixShader** from the Shader Packs menu.

---

## 💻 Requirements

* Minecraft **1.21.x+**
* Iris Shaders (Recommended) or OptiFine
* GPU with OpenGL support
